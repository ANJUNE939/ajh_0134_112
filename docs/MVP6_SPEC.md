# MVP6 SPEC

## 목표

5차에서 분리한 구직자/기업 페이지 구조에 회원가입, 로그인, 역할별 대시보드, 개인회원 유료 서비스를 추가한다.

## 회원 유형

### 개인회원

외국인 구직자. 필수적으로 국적, 비자, 외국인등록증 보유 여부, 시간제 취업 허가 상태, 학교, 전공, 언어 수준, 희망 근무형태, 근무 가능시간을 입력한다.

### 기업회원

외국인 인재 채용 기업. 기업명, 담당자, 연락처, 업종, 근무지, 채용 계획, 행정 지원 필요 여부를 입력한다.

## 유료 서비스

1. 행정 패스트트랙: 빠른 행정처리 요청, 서류 체크리스트, 행정사 파트너 우선 연결 데모
2. 매칭 부스트: 빠른 구직 매칭 요청, 이력서 우선 검토, 추천 공고 큐레이션 데모

## 법적 주의

실제 구직자 대상 유료 매칭 상품은 직업소개사업/수수료 관련 법적 검토가 필요하다. 현재는 결제 데모 기록만 생성한다.
